3-D Display of Earthquakes at Parkfield

  • The red lines represent the trace of the San Andreas fault (at the surface).
  • Most earthquakes are shown as white points.
  • Earthquakes with fault plane solutions are represented by blue rectangle, oriented in a selected fault plane
  • The image may be rotated, zoomed, and set spinning with your mouse.
Instructions
Operation Effect
Drag mouse (left mouse button pressed) Rotate about an axis in the picture
Release mouse while dragging Spin about an axis in the picture
Click mouse Stop spinning
Drag mouse vertically with SHIFT key pressed Zoom
Drag mouse horizontally with SHIFT key pressed Rotate about axis perpendicular to screen
Drag mouse vertically with CONTROL key pressed changing focal length
Drag mouse horizontally with CONTROL key pressed changing strength of stereo effect
Drag mouse vertically with META (ALT) key (or right mouse button) pressed stripping parts of the graphics
“o” key printing parameter settings to the Java console (aka Java messages window)
“s” key toggle between single picture, stereo pictures for diverge fusing and stereo pictures for cross fusing
HOME key restore original perspective
